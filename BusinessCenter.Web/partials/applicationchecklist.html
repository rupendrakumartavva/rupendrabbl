Application Checklist

Help With This Page (Click Here)

The Application Checklist page displays and captures a tailored list of the information and documentation you will need to submit to complete Basic Business License application.

  • The Application Checklist page is the user's "home page" while the required information for your BBL application is collected.
  • You can save an incomplete Checklist and resume work on it later. You can return to your Checklist from the My BBLs Page using the link

As valid information is entered, the Checklist will automatically acknowledge it by "checking off" the appropriate item on the Checklist.

Once you have completed the Application Checklist and all items are "checked off," the grey button will turn green and become active for you to advance to the next step.

Navigation to Data Entry Screens

  • Review Pre-Application Questions answers you provided using the link Review Your Answers to the Pre-Applications Questions (Here) .
    • *Note* you will be unable to alter the Pre-Application information at this point.
    • If you notice an error in an answer to a Pre-Application Question:
    • 1.   Return to the My Basic Business License Page
    • 2.   Select the button on the erroneous record
    • 3.    Start a new application with the button where corrections can be made.
  • Access the data entry fields to provide using the hyperlinks provided
    • Select the Enter Information about Your Business (Here) link to enter required information about the location and structure of your business.
    • Select the Upload Supporting Documents (Here) link to upload the required Supporting Documents.
    • Select the Enter Tax and Revenue Information (Here) link to provide required information and to complete the Clean Hands Attestation on the day you pay for and submit your Application. If you save and return to your Draft Application on another day, you will need to check and sign the Attestation form again.

You can save and exit at any time using the button.

If you wish to download a copy of the page as a PDF, you must select the button.

When the Checklist is Complete:

  • All items listed will display a indicating that the information has been accurately provided
  • The button at the bottom of the page will turn green and become active.

If you need additional help, review the My BBLs User Guide or contact the DCRA Help Desk at (202) 442-4400.
