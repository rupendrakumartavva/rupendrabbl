This page provides a read-only review the Pre-Application questions and answers provided to create the Checklist for a Basic Business License application.

Review the previously entered information. Edits to Pre-Application answers cannot be made to this application.

If you notice an error in an answer to a Pre-Application Question, a new Basic Business License must be started:

1.    Remove the erroneous Checklist:

  • Select the button to return to your My BBLs page.
  • Select the button on the erroneous record.

2.    Initiate a new BBL application:

  • Select the to begin a new application, where corrections can be made.

If you need additional help, review the My BBLs User Guide or contact the DCRA Help Desk at (202) 442-4400.
